Satisfactory Tools

A modified fork of the original Satisfactory Tools production planner and codex.

This deployment runs a modified fork of the original Satisfactory Tools project created by greeny, with help from Lucek and other contributors.

Our public fork lives at github.com/coyote-linux/SatisfactoryTools. The original upstream repository remains available at github.com/greeny/SatisfactoryTools.

This fork is maintained to keep the planner available at ficsit.spugnort.com while carrying forward deployment and gameplay updates in our modified codebase.

If you want the original project community, you can still join the original Discord server and follow updates from the upstream project.
